Important change:
Use scrollSnapStop: "normal" first, NOT "always".

Reason:
scroll-snap-stop: always can make fast flicks feel trapped, because it may force the scroll to stop at every slide. For an iPhone Photos-like feel, users should be able to flick naturally. Start with normal. Only test "always" if fast swipes skip too many images.

Initial position:
When fullscreen opens, set the scroll position instantly:

useEffect(() => {
  if (!isFullscreen) return;

  const el = scrollerRef.current;
  if (!el) return;

  requestAnimationFrame(() => {
    el.scrollTo({
      left: initialIndex * el.clientWidth,
      behavior: "auto",
    });
    setIndex(initialIndex);
  });
}, [isFullscreen, initialIndex]);

Index tracking:
Use a requestAnimationFrame-throttled scroll listener. This is fine and lightweight because it only calculates scrollLeft / clientWidth. Do not use IntersectionObserver here; it can lag behind native scrolling and is less direct for a paging viewer.

useEffect(() => {
  const el = scrollerRef.current;
  if (!el || !isFullscreen) return;

  let raf = 0;

  const onScroll = () => {
    if (raf) return;

    raf = requestAnimationFrame(() => {
      raf = 0;

      const width = el.clientWidth || 1;
      const nextIndex = Math.round(el.scrollLeft / width);

      if (nextIndex >= 0 && nextIndex < images.length) {
        setIndex((current) => (current === nextIndex ? current : nextIndex));
      }
    });
  };

  el.addEventListener("scroll", onScroll, { passive: true });

  return () => {
    el.removeEventListener("scroll", onScroll);
    if (raf) cancelAnimationFrame(raf);
  };
}, [isFullscreen, images.length]);

Arrows and dots:
Use native scrollTo:

const scrollToIndex = useCallback((targetIndex: number, behavior: ScrollBehavior = "smooth") => {
  const el = scrollerRef.current;
  if (!el) return;

  const clampedIndex = Math.max(0, Math.min(targetIndex, images.length - 1));

  el.scrollTo({
    left: clampedIndex * el.clientWidth,
    behavior,
  });

  setIndex(clampedIndex);
}, [images.length]);

For arrows/dots:
scrollToIndex(idx, "smooth")

For opening/re-aligning:
scrollToIndex(initialIndex, "auto")

Resize/orientation change:
Re-align instantly to current index:

useEffect(() => {
  if (!isFullscreen) return;

  const realign = () => {
    const el = scrollerRef.current;
    if (!el) return;

    el.scrollTo({
      left: index * el.clientWidth,
      behavior: "auto",
    });
  };

  window.addEventListener("resize", realign);
  window.addEventListener("orientationchange", realign);

  return () => {
    window.removeEventListener("resize", realign);
    window.removeEventListener("orientationchange", realign);
  };
}, [isFullscreen, index]);
